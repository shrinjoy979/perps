# Perps

A decentralized perpetual futures trading application built with modern web technologies. Users can open long/short positions, manage leverage, monitor PnL, and trade seamlessly through a clean and responsive interface.

## ✨ Features

- 🔐 Wallet authentication
- 📈 Long & Short positions
- ⚡ Adjustable leverage
- 💰 Real-time PnL calculation
- 📊 Position history
- 📉 Live market prices
- 🎯 Stop Loss & Take Profit (optional)
- 📱 Responsive UI

## 🛠️ Tech Stack

### Frontend
- React
- TypeScript

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/perps.git
cd perps
```

### Install dependencies

```bash
npm install
```

### Configure environment

Create a `.env` file.

```env
DATABASE_URL=
RPC_URL=
JWT_SECRET=
```

### Run the development server

```bash
npm run dev
```
